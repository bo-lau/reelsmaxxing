# Reelsmaxxing

Do you even love me if you're not watching all the IG reels I sent to you? 

## Installation

1. Open VSCode/Cursor
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
3. Type "Install from VSIX" and select it
4. Choose the `reelsmaxxing-0.0.1.vsix` file
5. Restart VSCode/Cursor

## Uninstall

1. Open Extensions in VSCode/Cursor
2. Find "Reelsmaxxing"
3. Click Uninstall
